# VISION.md

## Exploration Modelling Platform — Vision

This platform aims to redefine how geological exploration data is processed, modelled, and understood.

At its core is a computational graph where data flows through transparent, composable nodes. Each transformation is explicit, reproducible, and inspectable. The system combines structured geoscience models with a flexible execution engine, enabling users to move seamlessly from raw field data to interpretable 3D models.

Artificial intelligence acts as an assistant, not a black box. It helps classify data, map schemas, and suggest workflows, while keeping all decisions visible and controllable by the user.

The platform is designed to:
- unify disparate exploration datasets into a canonical structure
- provide real-time and incremental modelling workflows
- support both local and cloud-based computation
- enable collaboration, validation, and auditability
- scale from early exploration to advanced modelling

The long-term goal is to create a system where:
- data ingestion is effortless
- modelling pipelines are intuitive and adaptive
- insights emerge quickly and can be trusted
- technical and non-technical stakeholders can interact with the same model

This is not just a visualisation tool or a data processor. It is a domain-specific computational environment for understanding the subsurface.
